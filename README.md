# Garmin Activity Tracker

Garmin Connectに記録した活動データ（ランニング等）を取得し、ローカルで一覧・サマリー表示する個人利用向けアプリ。

> 設計の詳細は[基本設計書](docs/00_目次.md)、開発時のセットアップ手順は[PLAN.md](PLAN.md)を参照。このドキュメントは**日常的にアプリを使う人向け**の使い方ガイド。

## できること

- 「同期」ボタンで、Garmin Connectから直近30日分・全アクティビティ種別のデータを取得してローカルに保存する
- 保存済みのアクティビティを日付降順の一覧で見る（日付・種別・距離・時間・ペース・平均心拍・カロリー）
- 直近7日間・直近28日間の合計走行距離・合計時間のサマリーを見る（ランニングのみ集計）

## 初回セットアップ（最初の1回だけ）

まだ一度もセットアップしていない場合は、[PLAN.md](PLAN.md)の「セットアップ手順」に従って、依存パッケージのインストールと`scripts/init_login.py`によるGarminアカウントへの初回ログインを行う。ここで生成されるトークンキャッシュ（`~/.garminconnect/`）があれば、以降は再ログイン不要。

## 日常の使い方

### 1. アプリを起動する

```powershell
# ターミナル1: バックエンド
cd garmin\backend
.\venv\Scripts\Activate.ps1
uvicorn app.main:app --reload --port 8000

# ターミナル2: フロントエンド
cd garmin\frontend
npm run dev
```

### 2. ブラウザで開く

`http://localhost:5173` を開く。

### 3. 同期する

画面右上の「同期」ボタンを押す。Garmin Connectから直近30日分のデータを取得し、一覧・サマリーが自動的に更新される。同期中はボタンが「同期中...」表示になる。

- 同期しなくても、前回同期済みのデータはそのまま一覧・サマリーに表示され続ける（アプリを開くたびにGarminへ問い合わせるわけではない）
- 何度でも押してよい（重複登録はされず、既存データが更新される）

### 4. 画面を見る

**サマリーブロック**（一覧の上部）:
```
直近7日間 合計距離 21.6km / 合計時間 2時間19分
直近28日間 合計距離 61.4km / 合計時間 6時間23分
※ランニングのみ集計
```
「本日を含む直近7日間・28日間」のローリング集計（暦週・暦月ではない）。記録が無い期間は「0.0km / 0時間0分」と表示される。

**一覧テーブル**: 日付・種別・距離・時間・ペース・平均心拍・カロリーの列。保存済みの全アクティビティ種別（ランニング以外も含む）が表示される。一覧が0件の場合は「アクティビティがありません。同期してください」と表示される。

## 既知の制限事項

- **サマリーは種別が"running"と完全一致するものだけを集計する**。トレイルランニング（`trail_running`）など"running"以外の種別は一覧には表示されるが、サマリーの合計距離・合計時間には含まれない。実際の運動量より少なく表示される場合があるので、種別列も見てトレイル等の活動が無いか確認するとよい（詳細・今後の対応方針は[DESIGN_REVIEW_LOG.md](DESIGN_REVIEW_LOG.md)の2026-07-18エントリ参照）
- 同期の取得範囲は直近30日固定（`SYNC_LOOKBACK_DAYS`で変更可能）。それより古いアクティビティは同期されない
- 個人利用前提のため、複数人でのアクセスやインターネット公開は想定していない

## うまくいかないとき

| 症状 | 対処 |
|---|---|
| 「同期」ボタンでエラーになる | `scripts/init_login.py`のトークンキャッシュが失効している可能性。`backend`ディレクトリで`python scripts\init_login.py`を再実行する |
| フロントエンドに何も表示されない・CORSエラーが出る | バックエンド（`http://localhost:8000`）が起動しているか確認。`backend/.env`の`CORS_ALLOW_ORIGIN`がフロントエンドのURLと一致しているか確認 |
| `init_login.py`でMFAコードを求められる | ターミナルにそのままコードを入力する（Garmin Connectアプリ等に届く6桁コード） |
