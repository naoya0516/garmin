# 怪我予防

## ACWR（急性:慢性ワークロード比）

トレーニング負荷の急激な変化と怪我リスクの関係を捉える指標として、スポーツ科学分野で広く紹介されてきた考え方。

```
急性負荷 = 直近1週間の負荷（走行距離・時間など）
慢性負荷 = 直近4週間程度の平均負荷
ACWR = 急性負荷 ÷ 慢性負荷
```

- ACWRが概ね**0.8〜1.3程度に収まる範囲**は「スイートスポット」として比較的安全とされ、**1.5を大きく超える**急激な負荷増加は怪我リスクが高まる目安として紹介されることが多い（Gabbettらの研究に基づく経験則）
- ただし、ACWRという指標自体の算出方法（急性側と慢性側に同じデータが重複して使われる統計的な問題など）に対して近年批判もあり、「絶対的な安全基準」ではなく「負荷の急激な変化に注意を促す目安の一つ」として扱うのが妥当

## ランナーに多い障害と主な要因

| 障害 | 概要 | 主な誘因 |
|---|---|---|
| 腸脛靭帯炎(ITBS) | 膝の外側の痛み | 走行量の急増、下り坂の多用、筋力の左右差 |
| シンスプリント | すねの内側の痛み | 硬い路面での走り込み、急激な距離増加、足部の衝撃吸収不足 |
| 足底筋膜炎 | 足裏・かかとの痛み | 長時間の立位・走行、ふくらはぎ・足底の柔軟性不足 |
| ランナー膝(膝蓋大腿疼痛症候群) | 膝前面の痛み | 大腿四頭筋・臀筋の筋力不足、フォームの崩れ |

いずれも単一の原因というより、**トレーニング負荷・フォーム・筋力・柔軟性・シューズ/路面**などが複合的に関与するとされる。

## 予防の基本原則

- **漸進的過負荷の原則**: 距離・強度・頻度のいずれも急激に増やさず、段階的に負荷を上げる（[02_トレーニング負荷とピリオダイゼーション](02_トレーニング負荷とピリオダイゼーション.md) の週10%ルールも参照）
- **筋力トレーニングの併用**: 特に臀筋・体幹の筋力はランニングフォームの安定に寄与するとされ、怪我予防目的での併用が推奨されることが多い
- **痛みへの早期対応**: 「違和感」の段階で走行量を調整することで、慢性的な障害への進行を防ぎやすい
- **休養日の確保**: 完全休養日を週に最低1日程度設けることが一般的に推奨される
