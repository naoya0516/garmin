# ランニング科学ナレッジベース

`garmin-coach` エージェントが機能提案を行う際に参照する、ランニング科学の基礎知識をまとめたもの。

## 位置づけ

- `feature-pipeline` スキルがコーチ呼び出し時にこのフォルダ配下の全ファイルを読み込み、プロンプトに全文注入する（`garmin-coach`自身はReadツールを持たないため、コードには一切触れられない境界を維持したまま知識だけを受け取る）
- ここに書かれているのは、スポーツ科学で広く合意されている**教科書的・基礎的なフレームワーク**。個別の最新研究や特殊なケースについては、コーチ自身がWebSearch/WebFetchで補完する前提
- 内容はClaudeの一般知識をベースに、いくつかの数値・公式はWeb検索で裏付けを取って作成した。特定の書籍・論文・コーチング資格カリキュラムに準拠したものではない
- 各ファイル末尾に医療的判断についての注記があるとおり、これは一般的な知見の要約であり、個別の医療的助言ではない

## 更新方針

新しいトピックを追加する場合は、このフォルダに `05_xxx.md` の形式でファイルを追加し、`00_README.md`のファイル一覧を更新する。`feature-pipeline`側の変更は不要（フォルダ配下を全件読み込む実装のため）。
